How The System Works

A compact publishing workflow designed for hands-on operation from one laptop.

01

Topic Discovery

Surfaces high-signal news and trend topics for new reel ideas.

02

Script Build

Creates short-form, neutral scripts with hook-first structure.

03

Video Assembly

Combines visuals, voice, subtitles, and vertical formatting.

04

Review + Refresh

Operator opens the local dashboard to inspect updated performance and history.

Core Capabilities

  • Trend-led topic research
  • Script generation with factual framing
  • Narration and vertical reel composition
  • Instagram-oriented performance tracking
  • Project microsite plus local dashboard workflow

Local Dashboard Flow

The dashboard itself is a polished static snapshot. Run one local refresh command on your laptop to sync Instagram insights, rebuild the snapshot, and then push the updated files.

Refresh command: npm run dashboard:refresh
Local microsite: http://localhost:3000/microsite.html
Public dashboard path: /ai-reel-agent/public/dashboard.html
