## 🚨 Troubleshooting

### If workflow doesn't run:
1. Check secrets are set correctly
2. Verify Railway URL is correct
3. Check webhook endpoint is accessible

### If Instagram posting fails:
1. Verify Instagram token is still valid
2. Check business account ID is numeric
3. Test with `/api/webhook/test-instagram-post` locally

## 📞 Support

If you need to:
- **Change schedule time:** Edit `.github/workflows/daily-reel.yml` (cron expression)

- **Stop scheduler:** Disable the workflow in Actions
